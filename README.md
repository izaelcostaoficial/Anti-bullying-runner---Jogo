# Anti-Bullying Runner Pro
Um jogo estilo Endless Runner 3D/Pseudo-3D responsivo.

## Estrutura do Projeto
- `index.html` : Estrutura principal, telas de menu, loja e HUD do jogo.
- `style.css` : Estilização moderna, cores vibrantes, responsividade mobile/desktop.
- `game.js` : O motor do jogo, lógica de movimentação (3 pistas), colisões, perguntas e placar.

## Como Executar
1. Coloque os 3 arquivos na mesma pasta.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
